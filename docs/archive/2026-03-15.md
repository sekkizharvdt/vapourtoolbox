## Changes Since Last Review (328 commits)

### Major Features Added

#### Thermal Engineering Module (Massive Expansion)

Evolved from basic calculators to a comprehensive MED design platform:

- **MED Plant Calculator**: Heat & mass balance (Phase 1), MED-TVC integration (Phase 2), equipment sizing with Dr. Rognoni reference comparisons (Phase 3)
- **MED-TVC reference projects page** with as-built design data from 3 real projects
- **Heat Transfer**: Consolidated heat duty, HTC, and heat exchanger into unified iterative design tool; evaporator and liquid-liquid modes; falling film evaporator; fouling/scaling; GOR calculator
- **TVC/MVC**: Thermo-vapor compressor, mechanical vapor compressor, desuperheating calculators
- **Pump & Fluid Systems**: Siphon sizing (inter-effect pipes), suction system designer, pump sizing, spray nozzle selection from CAT75HYD catalogue
- **Vacuum Systems**: Vacuum breaker sizing for MED plants, LRVP sizing, NCG properties calculator
- **Strainer sizing** with batch mode and auto-populated fluid properties
- **Infrastructure**: PDF report exports for 5+ calculators, save/load across 9+ calculators, batch mode, Excel export
- **Mesh size selector** with neighbor comparison table

#### Accounting Module (14+ Features)

- **Direct Receipts** — new transaction type with Chart of Accounts integration
- **Fixed Asset Registry** — 4 phases: foundation, detail page, depreciation engine, bill-to-asset integration (41 new tests)
- **Tally-style Enter-key navigation** — `useTallyKeyboard` hook wired to 20+ dialogs
- **Soft delete + Trash page** for transactions
- **Excel/CSV export** for all transaction list pages
- **Void action** for customer invoices
- **Opening balance allocation** for payments
- **TDS rate categories**
- **GST summary report**
- **Data Health dashboard** for fixing accounting data issues
- **GL drill-down** for reports with P&L transaction drill-down
- **Stale payments tracker**

#### Flow Module (Complete Redesign)

- Landing page with card-based hub
- Tasks page with list and create dialog
- Inbox page with filter chips (reuses TaskCard)
- Team Board showing active tasks per member
- Meeting Minutes with two-step creation and batch finalization

#### Procurement Module (10+ Features)

- CSV/PDF export and soft delete across all list pages
- RFQ state machine with PO_PROCESSED status
- Commercial terms to offer comparison

- Replace GRN Create Bill with Send to Accounting workflow

#### Documents Module (7 Features)

- Workflow automation and approvals

#### Materials Module

#### Proposals Module (Major Refactor)
